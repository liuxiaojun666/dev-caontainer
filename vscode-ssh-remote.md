### 配置ssh

生成公钥和私钥
```bash
ssh-keygen -t rsa -C "
```

将公钥复制到服务器上
```bash
ssh-copy-id -i ~/.ssh/id_rsa.pub
```

服务器上配置
```bash
vim /etc/ssh/sshd_config
```
修改配置
```bash
PubkeyAuthentication yes
AuthorizedKeysFile .ssh/authorized_keys
```

重启ssh服务
```bash
service sshd restart
```

将本地的公钥复制到服务器上的authorized_keys文件中
```bash
vim ～/.ssh/authorized_keys
```
